# Phase-RB campaign working files

Raw gather claims and refuter verdicts land here AS THEY ARRIVE and are
committed immediately — insurance against session loss mid-chunk (the
session scratchpad does not survive a hard reset). When a chunk's
synthesized section lands in `research/ambient-flows.md` or the basin
files, that chunk's WIP files are deleted in the same commit (the
synthesis supersedes them; the verification record preserves the tallies).
Resume rule: `research/rb-campaign.md` names the current chunk; any WIP
files here belong to it.
